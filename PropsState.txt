# Props와 State
-Props: 'Properties'로 리액트 컴포넌트의 특성, 이를 통해 부모 컴포넌트가 자식 컴포넌트에 데이터 전달 가능.
부모 컴포넌트로부터 전달받는 데이터이고 해당 컴포넌트의 특성이므로 자식 컴포넌트에서는 변경 불가능.
-State: 리액트 컴포넌트의 현재 상태, 변경 가능.
한 컴포넌트 안에서 유동적인 데이터를 다룰 때에 사용되며, 컴포넌트 안에서 데이터 변경 가능.
